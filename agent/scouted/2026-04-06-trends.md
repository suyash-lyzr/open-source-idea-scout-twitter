## 🕳️ Gaps Identified (for idea generation)

### Gap 1: No tool to auto-generate DESIGN.md files
- `awesome-design-md` has 13.6k stars but it's just a static collection
- No tool crawls your existing codebase/app and auto-generates a DESIGN.md
- Existing design system tools (Storybook, Zeroheight) are heavy and not AI-agent-focused
- **Opportunity**: CLI tool that analyzes a repo and outputs a DESIGN.md

### Gap 2: No good open-source LLM token cost tracker
- Codex pricing change is forcing devs to understand per-token costs
- Existing tools: toktrack (78⭐), OpenLLM-Monitor (35⭐), llm-cost-dashboard (2⭐)
- All are tiny, incomplete, or not maintained
- **Opportunity**: A proper, polished OSS token cost dashboard proxy

### Gap 3: No AI agent decision audit log
- HN's #1 story (680pts, 461 comments) is about "comfortable drift" — devs not understanding what AI agents are doing
- Zero OSS tools for logging/auditing AI agent decisions in plain English
- **Opportunity**: A lightweight audit middleware that logs every AI agent action with reasoning
